
# perf-test-cli

CLI for performance benchmarking

## Installation
### From Source

```bash
# Clone the repository
git clone https://github.com/user/perf-test-cli
cd perf-test-cli

# Build and install
cargo build --release
cargo install --path .
```

### Using Cargo

```bash
cargo install perf-test-cli
```
