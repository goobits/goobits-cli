
# nested-demo

Demonstration of deep nested command capabilities

## Installation
### From Source

```bash
# Clone the repository
git clone https://github.com/user/nested-demo
cd nested-demo

# Build and install
cargo build --release
cargo install --path .
```

### Using Cargo

```bash
cargo install nested-demo
```
