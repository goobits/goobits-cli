
# demo-rust-cli

A sample Rust CLI built with Goobits

## Installation
### From Source

```bash
# Clone the repository
git clone https://github.com/user/demo-rust-cli
cd demo-rust-cli

# Build and install
cargo build --release
cargo install --path .
```

### Using Cargo

```bash
cargo install demo-rust-cli
```
